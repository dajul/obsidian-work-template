# Meetings

- Meetings folder: [[Meetings]]
- Tracking base: [[Bases/Meetings.base]]
- Meeting template: [[Templates/Meeting Note]]

## How to use

1. Create one note per meeting in `Meetings`
2. Capture decisions and action items in that same note
3. Set `follow_up: true` if follow-up is still needed

## Naming convention

- Use `YYYY-MM-DD Team Sync`
- Examples: `2026-03-20 Team Sync`, `2026-03-20 1:1 Manager`, `2026-03-20 Sprint Retro`

## Open meeting actions

```tasks
not done
path includes Meetings
sort by scheduled
sort by due
sort by priority
```

## Actions by meeting

```tasks
not done
path includes Meetings
group by filename
sort by path
```
