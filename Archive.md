# Archive

- Folder: [[Archive]]
- Review base: [[Bases/Archive.base]]

## Best way to use the archive

1. Keep active notes in `Notes/`
2. Change `status` to `archived` before moving a note here
3. Move inactive but useful notes into `Archive/`
4. Use this area for retrieval, not active work

## Open tasks still inside archive

```tasks
not done
path includes Archive
sort by scheduled
sort by due
sort by priority
```
