- Review base: [[Bases/Notes.base]]

## Best way to use notes

1. Create a new note inside `Notes/`
2. Use `Templates/Note`
3. Move cleaned-up information there from `Inbox/`, `Daily/`, or `Meetings/`
4. Keep reusable references, decisions, and write-ups there
5. Review older notes through the base instead of tabs

## Open note tasks
