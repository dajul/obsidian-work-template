# Inbox

- Folder: [[Inbox]]
- Review base: [[Bases/Inbox.base]]
- Template: [[Templates/Inbox Note]]

## Best way to use inbox notes

1. Create a new note inside `Inbox/`
2. Name it `YYYY-MM-DD HHmm - topic`
3. Apply `Templates/Inbox Note`
4. Keep only the note you are actively editing open
5. Review older notes through the base instead of tabs
6. Delete the note or change `status` to `archived` when done

## Open inbox tasks

```tasks
not done
path includes Inbox
sort by scheduled
sort by due
sort by priority
```
